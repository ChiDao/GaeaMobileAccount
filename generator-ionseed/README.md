iontest
==========

##前提条件

```bash
npm install yo -g
```

##安装

```bash
npm install
```

```bash
npm link
```

##使用

```bash
cd ../client
```

####建立Provider

```bash
yo ionseed:provider
```

####建立Service

```bash
yo ionseed:service
```

####建立单独页面

```bash
yo ionseed:page
```

####建立列表详情页面：

```bash
yo ionseed:listdetail    
```
